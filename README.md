## RAS RC-Car has been used RaspberryPi GPIO PIN

* **Motor Drive** : SN754410NE 
* **RaspberryPi GPIO**   
   - Motor A : GPIO 4, GPIO 27
   - Motor B : GPIO 17, GPIO 22
